== Description ==
Myanmar daily foreign exchange (forex) rates WordPress plugin. This plugin gets daily currency exchange rates from Central Bank of Myanmar (CBM).

You can use

1. plugin setting page inside WordPress dashboard
2. the shortcode
3. or the template tag

to display the currency exchange rate on you WordPress site.

== Installation ==

1. Search "Myanmar Currency Exchange Rates" in WordPress plugin installation interface
2. Click "Intsall" and then activate it
3. Go to Appearance » Widgets and find the "Myanmar Currency Exchange Rates" widget
4. For the configuration interface, go to Settings » Myanmar Currency Exchange Rates menu

== Frequently Asked Questions ==

= Can I add the exchange rates table inside the page? =

Yes, you can use `[mm_fxrates]` shortcode to display the exchange rates table on any page.

== Screenshots ==

1. Go to plugin installation screen and search for "Myanmar Currency Exchange Rates" or "Myanmar Exchange Rates". You will see the plugin as shown in the screenshots. Go ahead and hit "Install Now" and "Activate" after that.
2. After installation, go to Appearance » Widgets to place the widget at your desire position.
3. The plugin comes with two display modes, "Normal" and "Compact". Choose as you prefer and choose the currencies to display. Then save the changes.
4. Now, visit your website to see that Myanmar Currency Exchange Rates widget will be neatly appear on the chosen location.
